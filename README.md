# CodeAlpha_Calculator

## Project Overview

This project is a responsive calculator developed using HTML, CSS, and JavaScript as part of the CodeAlpha Web Development Internship.

## Features

* Addition, subtraction, multiplication, and division operations
* Clear and delete functionality
* Real-time display updates
* Keyboard support
* Responsive design
* Modern user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* TypeScript

## How to Run

1. Download or clone the repository.
2. Open terminal "npm install" "npmrun dev"
